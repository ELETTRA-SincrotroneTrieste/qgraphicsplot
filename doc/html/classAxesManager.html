◆ getAssociatedAxis()

ScaleItem * AxesManager::getAssociatedAxis ( ScaleItem::Id  otherAxisId)

given an axis with otherAxisId as id, this method returns the associated axis

Note

If one axis is shared through different couples, (for instance an x axis common between two distinct y axes), then the associated axis in the first couple in the list is returned.

Returns
the axis associated to the axis identified by otherAxisId.

Axes are coupled in two. This method returns the one coupled with otherAxisId
