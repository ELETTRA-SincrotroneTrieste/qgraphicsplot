Detailed Description

This class is created and managed by the PlotSceneWidget.

Configuration
The legend can be configured by changing its zValue (default value is 100), by filling its background and drawing a black bounding rect around (both disabled by default.
Accessing the LegendItem.
Use PlotSceneWidget::legendItem()
Curve aliases.
The legend is able to show an alias instead of the complete source name of the curve. Use setCurveAlias to setup an alias for each curve. setCurveAlias can be invoked before or after TPlotSceneWidget::setSourcesList
