\doxysection{Curve\+Item Class Reference}
\label{classCurveItem}\index{CurveItem@{CurveItem}}


A Q\+Graphics\+Object that allows an implementation of the \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} to draw on a \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}. If an \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} (or more) is installed, then the draw method of the \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} is invoked instead of the default paint method.

\doxysubsection{Detailed Description}
\begin{DoxySeeAlso}{See also}
\doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} 

\doxyref{Scene\+Curve}{p.}{classSceneCurve} 

\doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}
\end{DoxySeeAlso}
The \doxyref{Curve\+Item}{p.}{classCurveItem} class calculates the necessary transformations that allow to correctly map the real data values to the plot canvas coordinates. It makes use of classes implementing the \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} interface in order to draw the data on the plot canvas.

\begin{DoxyParagraph}{Example}
Read the \doxyref{Scene\+Curve}{p.}{classSceneCurve} documentation for an example.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Item painters ownership}
The \doxyref{Curve\+Item}{p.}{classCurveItem} takes the ownership of every installed \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} implementation instance. This means that all the installed \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} objects are deleted on \doxyref{Curve\+Item}{p.}{classCurveItem} deletion.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Curve\+Item ownership}
If you call Plot\+Scene\+Widget\+::remove\+Curve, then the associated \doxyref{Curve\+Item}{p.}{classCurveItem} is deleted. If you delete a \doxyref{Scene\+Curve}{p.}{classSceneCurve} without invoking Plot\+Scene\+Widget\+::remove\+Curve (bad), then the associated \doxyref{Curve\+Item}{p.}{classCurveItem} is deleted again for you. This means that once installed on a \doxyref{Scene\+Curve}{p.}{classSceneCurve}, the ownership of the \doxyref{Curve\+Item}{p.}{classCurveItem} is taken by the \doxyref{Scene\+Curve}{p.}{classSceneCurve}. With the term {\itshape install} we mean the
\end{DoxyParagraph}
