\subsection{Detailed Description}
\subsubsection*{template$<$class Property\+Manager$>$\\*
class Qt\+Abstract\+Editor\+Factory$<$ Property\+Manager $>$}

An editor factory is a class that is able to create an editing widget of a specified type (e.\+g. line edits or comboboxes) for a given \doxyref{Qt\+Property}{p.}{classQtProperty} object, and it is used in conjunction with the \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} and \doxyref{Qt\+Abstract\+Property\+Browser}{p.}{classQtAbstractPropertyBrowser} classes.

Note that the \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} functions are using the Property\+Manager template argument class which can be any \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} subclass. For example\+:


\begin{DoxyCode}
QtSpinBoxFactory *factory;
QSet<QtIntPropertyManager *> managers = factory->propertyManagers();
\end{DoxyCode}


Note that \doxyref{Qt\+Spin\+Box\+Factory}{p.}{classQtSpinBoxFactory} by definition creates editing widgets {\itshape only} for properties created by \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager}.

When using a property browser widget, the properties are created and managed by implementations of the \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} class. To ensure that the properties\textquotesingle{} values will be displayed using suitable editing widgets, the managers are associated with objects of \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} subclasses. The property browser will use these associations to determine which factories it should use to create the preferred editing widgets.

A \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} object is capable of producing editors for several property managers at the same time. To create an association between this factory and a given manager, use the \doxyref{add\+Property\+Manager()}{p.}{classQtAbstractEditorFactory_af32e5e5341a42399841c97c0e82f1f5c} function. Use the \doxyref{remove\+Property\+Manager()}{p.}{classQtAbstractEditorFactory_a287d46dfa42baa38c8d2d34afc697432} function to make this factory stop producing editors for a given property manager. Use the \doxyref{property\+Managers()}{p.}{classQtAbstractEditorFactory_a62c833c60837abdc41c4a8e416b96b63} function to retrieve the set of managers currently associated with this factory.

Several ready-\/made implementations of the \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} class are available\+:

\doxyref{Qt\+Check\+Box\+Factory}{p.}{classQtCheckBoxFactory}  \doxyref{Qt\+Date\+Edit\+Factory}{p.}{classQtDateEditFactory}  \doxyref{Qt\+Date\+Time\+Edit\+Factory}{p.}{classQtDateTimeEditFactory}  \doxyref{Qt\+Double\+Spin\+Box\+Factory}{p.}{classQtDoubleSpinBoxFactory}  \doxyref{Qt\+Enum\+Editor\+Factory}{p.}{classQtEnumEditorFactory}  \doxyref{Qt\+Line\+Edit\+Factory}{p.}{classQtLineEditFactory}  \doxyref{Qt\+Scroll\+Bar\+Factory}{p.}{classQtScrollBarFactory}  \doxyref{Qt\+Slider\+Factory}{p.}{classQtSliderFactory}  \doxyref{Qt\+Spin\+Box\+Factory}{p.}{classQtSpinBoxFactory}  \doxyref{Qt\+Time\+Edit\+Factory}{p.}{classQtTimeEditFactory}  \doxyref{Qt\+Variant\+Editor\+Factory}{p.}{classQtVariantEditorFactory} 

When deriving from the \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} class, several pure virtual functions must be implemented\+: the \doxyref{connect\+Property\+Manager()}{p.}{classQtAbstractEditorFactory_acc1d86b1041a3ff043e8aef756316081} function is used by the factory to connect to the given manager\textquotesingle{}s signals, the \doxyref{create\+Editor()}{p.}{classQtAbstractEditorFactory_a5386f2a9e2d641ce88feaaa5f78fa33a} function is supposed to create an editor for the given property controlled by the given manager, and finally the \doxyref{disconnect\+Property\+Manager()}{p.}{classQtAbstractEditorFactory_a39ea7ae0ca4bc7a56b5483f0a16f94ae} function is used by the factory to disconnect from the specified manager\textquotesingle{}s signals.
