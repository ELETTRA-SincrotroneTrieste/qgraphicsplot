\subsection{Detailed Description}
A manager can create and manage properties of a given type, and is used in conjunction with the \doxyref{Qt\+Abstract\+Property\+Browser}{p.}{classQtAbstractPropertyBrowser} class.

When using a property browser widget, the properties are created and managed by implementations of the \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} class. To ensure that the properties\textquotesingle{} values will be displayed using suitable editing widgets, the managers are associated with objects of \doxyref{Qt\+Abstract\+Editor\+Factory}{p.}{classQtAbstractEditorFactory} subclasses. The property browser will use these associations to determine which factories it should use to create the preferred editing widgets.

The \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} class provides common functionality like creating a property using the \doxyref{add\+Property()}{p.}{classQtAbstractPropertyManager_abc5b894d6b9ba0441edebe561ca9f2f3} function, and retrieving the properties created by the manager using the \doxyref{properties()}{p.}{classQtAbstractPropertyManager_af10a7f87a398b2b49ece913baf46b1ec} function. The class also provides signals that are emitted when the manager\textquotesingle{}s properties change\+: \doxyref{property\+Inserted()}{p.}{classQtAbstractPropertyManager_a3e23d91d77fd1ad3657f5e12efd83839}, \doxyref{property\+Removed()}{p.}{classQtAbstractPropertyManager_ad80c2fd295b32b398c90bb85e145d4c0}, \doxyref{property\+Changed()}{p.}{classQtAbstractPropertyManager_a4dd203a9014a26e141b8a8f6b8c5e22d} and \doxyref{property\+Destroyed()}{p.}{classQtAbstractPropertyManager_a2aefea1236ae8c6451d58a87db3be309}.

\doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} subclasses are supposed to provide their own type specific A\+PI. Note that several ready-\/made implementations are available\+:

\doxyref{Qt\+Bool\+Property\+Manager}{p.}{classQtBoolPropertyManager}  \doxyref{Qt\+Color\+Property\+Manager}{p.}{classQtColorPropertyManager}  \doxyref{Qt\+Date\+Property\+Manager}{p.}{classQtDatePropertyManager}  \doxyref{Qt\+Date\+Time\+Property\+Manager}{p.}{classQtDateTimePropertyManager}  \doxyref{Qt\+Double\+Property\+Manager}{p.}{classQtDoublePropertyManager}  \doxyref{Qt\+Enum\+Property\+Manager}{p.}{classQtEnumPropertyManager}  \doxyref{Qt\+Flag\+Property\+Manager}{p.}{classQtFlagPropertyManager}  \doxyref{Qt\+Font\+Property\+Manager}{p.}{classQtFontPropertyManager}  \doxyref{Qt\+Group\+Property\+Manager}{p.}{classQtGroupPropertyManager}  \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager}  \doxyref{Qt\+Point\+Property\+Manager}{p.}{classQtPointPropertyManager}  \doxyref{Qt\+Rect\+Property\+Manager}{p.}{classQtRectPropertyManager}  \doxyref{Qt\+Size\+Property\+Manager}{p.}{classQtSizePropertyManager}  \doxyref{Qt\+Size\+Policy\+Property\+Manager}{p.}{classQtSizePolicyPropertyManager}  \doxyref{Qt\+String\+Property\+Manager}{p.}{classQtStringPropertyManager}  \doxyref{Qt\+Time\+Property\+Manager}{p.}{classQtTimePropertyManager}  \doxyref{Qt\+Variant\+Property\+Manager}{p.}{classQtVariantPropertyManager}
