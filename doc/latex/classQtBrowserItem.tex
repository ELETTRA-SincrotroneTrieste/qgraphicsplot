\subsection{Detailed Description}
Browser items are created whenever a \doxyref{Qt\+Property}{p.}{classQtProperty} is inserted to the property browser. A \doxyref{Qt\+Browser\+Item}{p.}{classQtBrowserItem} uniquely identifies a browser\textquotesingle{}s item. Thus, if the same \doxyref{Qt\+Property}{p.}{classQtProperty} is inserted multiple times, each occurrence gets its own unique \doxyref{Qt\+Browser\+Item}{p.}{classQtBrowserItem}. The items are owned by \doxyref{Qt\+Abstract\+Property\+Browser}{p.}{classQtAbstractPropertyBrowser} and automatically deleted when they are removed from the browser.

You can traverse a browser\textquotesingle{}s properties by calling \doxyref{parent()}{p.}{classQtBrowserItem_a4ac772c5b9b4601bd85a35ff06a8058f} and \doxyref{children()}{p.}{classQtBrowserItem_a19aec2ce6a6eeaf34eac0b36fbb33ede}. The property and the browser associated with an item are available as \doxyref{property()}{p.}{classQtBrowserItem_a8cfce687bc55a33defacbd321b3a77d2} and \doxyref{browser()}{p.}{classQtBrowserItem_aadd4f46f9df050fcb5fbcf4e8f769924}.
