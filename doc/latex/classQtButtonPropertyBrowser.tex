\subsection{Detailed Description}
A property browser is a widget that enables the user to edit a given set of properties. Each property is represented by a label specifying the property\textquotesingle{}s name, and an editing widget (e.\+g. a line edit or a combobox) holding its value. A property can have zero or more subproperties.

\doxyref{Qt\+Button\+Property\+Browser}{p.}{classQtButtonPropertyBrowser} provides drop down button for all nested properties, i.\+e. subproperties are enclosed by a container associated with the drop down button. The parent property\textquotesingle{}s name is displayed as button text. For example\+:

Use the \doxyref{Qt\+Abstract\+Property\+Browser}{p.}{classQtAbstractPropertyBrowser} A\+PI to add, insert and remove properties from an instance of the \doxyref{Qt\+Button\+Property\+Browser}{p.}{classQtButtonPropertyBrowser} class. The properties themselves are created and managed by implementations of the \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} class.
