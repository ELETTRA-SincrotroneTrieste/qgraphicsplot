A char\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtCharPropertyManager_a13a66db5ca346bae773b4e6ce1f87c1c} function, and set using the \doxyref{set\+Value()}{p.}{classQtCharPropertyManager_a941031804f25b000cfa753621500ac05} slot.
