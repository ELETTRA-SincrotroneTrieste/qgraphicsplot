\subsection{Detailed Description}
A color property has nested {\itshape red}, {\itshape green} and {\itshape blue} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtColorPropertyManager_ad3298ba9dd66d39449ab09dd6f41e402} function, and set using the \doxyref{set\+Value()}{p.}{classQtColorPropertyManager_ab18ff6b769efadd18ec2f56b5ec396bf} slot.

The subproperties are created by a \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtColorPropertyManager_a67b7ecc3e0b2e3fd30410fe0b3f0d108} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

In addition, \doxyref{Qt\+Color\+Property\+Manager}{p.}{classQtColorPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtColorPropertyManager_ad13c4dd85cb2e9f9b269f54a09781969} signal which is emitted whenever a property created by this manager changes.
