\subsection{Detailed Description}
A cursor property has a current value which can be retrieved using the \doxyref{value()}{p.}{classQtCursorPropertyManager_a47ff7d9abf3fbd82d95870d94455c2f0} function, and set using the \doxyref{set\+Value()}{p.}{classQtCursorPropertyManager_a0079c81ab012781d1d8f495ed9987a2d} slot. In addition, \doxyref{Qt\+Cursor\+Property\+Manager}{p.}{classQtCursorPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtCursorPropertyManager_aa770949898717977040fc1bebde78719} signal which is emitted whenever a property created by this manager changes.
