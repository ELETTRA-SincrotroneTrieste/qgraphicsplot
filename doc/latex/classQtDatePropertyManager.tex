\subsection{Detailed Description}
A date property has a current value, and a range specifying the valid dates. The range is defined by a minimum and a maximum value.

The property\textquotesingle{}s values can be retrieved using the \doxyref{minimum()}{p.}{classQtDatePropertyManager_a05ede99c3218e11603d6d169ad8a526c}, \doxyref{maximum()}{p.}{classQtDatePropertyManager_a541ff2359a97d5dbdf90c98da56840e9} and \doxyref{value()}{p.}{classQtDatePropertyManager_afaebda214465daa9a67c88ea0b7cb6e3} functions, and can be set using the \doxyref{set\+Minimum()}{p.}{classQtDatePropertyManager_afb0c947e2febe8d8ecae0701321d6a31}, \doxyref{set\+Maximum()}{p.}{classQtDatePropertyManager_a58a38b2da9b4540e967036706d8fc4d7} and \doxyref{set\+Value()}{p.}{classQtDatePropertyManager_ade19a28fd11b45fbfbc407fbad2b6331} slots. Alternatively, the range can be defined in one go using the \doxyref{set\+Range()}{p.}{classQtDatePropertyManager_a18aa426b0d4f0779f2dec1b5f0a4e287} slot.

In addition, \doxyref{Qt\+Date\+Property\+Manager}{p.}{classQtDatePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtDatePropertyManager_a6ea1490e9e268a72fe529800f5fdac87} signal which is emitted whenever a property created by this manager changes, and the \doxyref{range\+Changed()}{p.}{classQtDatePropertyManager_a2dc434ea66104f50ff15a645bea4c679} signal which is emitted whenever such a property changes its range of valid dates.
