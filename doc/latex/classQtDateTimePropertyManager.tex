\subsection{Detailed Description}
A date and time property has a current value which can be retrieved using the \doxyref{value()}{p.}{classQtDateTimePropertyManager_abf5e0ee2d57fe0bb3d6a12264fd0f9b4} function, and set using the \doxyref{set\+Value()}{p.}{classQtDateTimePropertyManager_ac9b9b353bdabc0ab25c3b3aaae54acba} slot. In addition, \doxyref{Qt\+Date\+Time\+Property\+Manager}{p.}{classQtDateTimePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtDateTimePropertyManager_ad9b2fdd107e0ec207110762a7f62e53d} signal which is emitted whenever a property created by this manager changes.
