\subsection{Detailed Description}
A double property has a current value, and a range specifying the valid values. The range is defined by a minimum and a maximum value.

The property\textquotesingle{}s value and range can be retrieved using the \doxyref{value()}{p.}{classQtDoublePropertyManager_a43660de019d315e365e93ddbfc8cefdb}, \doxyref{minimum()}{p.}{classQtDoublePropertyManager_a7ade0f50e0ef8cfcd34cc87edfbfe241} and \doxyref{maximum()}{p.}{classQtDoublePropertyManager_a0cfaa2ad37770f4df2489202c70869e8} functions, and can be set using the \doxyref{set\+Value()}{p.}{classQtDoublePropertyManager_ac38dda79cdc700a6484d6a6cdd234d75}, \doxyref{set\+Minimum()}{p.}{classQtDoublePropertyManager_ac303052e95e61e4bbce4e0b70efe35eb} and \doxyref{set\+Maximum()}{p.}{classQtDoublePropertyManager_a5f5bde901cbcab806e8b75d05c0230e4} slots. Alternatively, the range can be defined in one go using the \doxyref{set\+Range()}{p.}{classQtDoublePropertyManager_af272ed9deea94559169f2553eb127007} slot.

In addition, \doxyref{Qt\+Double\+Property\+Manager}{p.}{classQtDoublePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtDoublePropertyManager_a3b088a6be8424a7aaed6b78df4d1c430} signal which is emitted whenever a property created by this manager changes, and the \doxyref{range\+Changed()}{p.}{classQtDoublePropertyManager_ac724186dcc4b2eea6598dace14a7d498} signal which is emitted whenever such a property changes its range of valid values.
