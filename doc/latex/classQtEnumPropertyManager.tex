\subsection{Detailed Description}
Each enum property has an associated list of enum names which can be retrieved using the \doxyref{enum\+Names()}{p.}{classQtEnumPropertyManager_a6019930db4d20d1f28babf34cc33c3d3} function, and set using the corresponding \doxyref{set\+Enum\+Names()}{p.}{classQtEnumPropertyManager_a6cdbee64ff4e2174e68e8a52acfd6a49} function. An enum property\textquotesingle{}s value is represented by an index in this list, and can be retrieved and set using the \doxyref{value()}{p.}{classQtEnumPropertyManager_a6ce3bf541ecb3a48a017b520488500c4} and \doxyref{set\+Value()}{p.}{classQtEnumPropertyManager_a2687a8ff63fdfe89290f330aa9f71ccb} slots respectively.

Each enum value can also have an associated icon. The mapping from values to icons can be set using the \doxyref{set\+Enum\+Icons()}{p.}{classQtEnumPropertyManager_a14a47ec5109c7a333cb83971aeb8f835} function and queried with the \doxyref{enum\+Icons()}{p.}{classQtEnumPropertyManager_a64a0637df02059a95b95c61c5f981fb9} function.

In addition, \doxyref{Qt\+Enum\+Property\+Manager}{p.}{classQtEnumPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtEnumPropertyManager_aa18c16533e9d538faa78411a4d1b3191} signal which is emitted whenever a property created by this manager changes. The \doxyref{enum\+Names\+Changed()}{p.}{classQtEnumPropertyManager_a479c11e32a02eda3461e9b573c79dde5} or \doxyref{enum\+Icons\+Changed()}{p.}{classQtEnumPropertyManager_a43e80c877ef4509d21dbb8a516d186e4} signal is emitted whenever the list of enum names or icons is altered.
