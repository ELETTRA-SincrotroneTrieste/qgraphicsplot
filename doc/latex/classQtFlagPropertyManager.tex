\subsection{Detailed Description}
Each flag property has an associated list of flag names which can be retrieved using the \doxyref{flag\+Names()}{p.}{classQtFlagPropertyManager_a765ffaee7976dcb1de6a494d22088dff} function, and set using the corresponding \doxyref{set\+Flag\+Names()}{p.}{classQtFlagPropertyManager_ac9a69f02f5b95265dde11182b22584b1} function.

The flag manager provides properties with nested boolean subproperties representing each flag, i.\+e. a flag property\textquotesingle{}s value is the binary combination of the subproperties\textquotesingle{} values. A property\textquotesingle{}s value can be retrieved and set using the \doxyref{value()}{p.}{classQtFlagPropertyManager_ad67ba0980ad4b64627948423eed0a672} and \doxyref{set\+Value()}{p.}{classQtFlagPropertyManager_aad7c51a700c985b1bbad8aa6cfcbf22a} slots respectively. The combination of flags is represented by single int value -\/ that\textquotesingle{}s why it\textquotesingle{}s possible to store up to 32 independent flags in one flag property.

The subproperties are created by a \doxyref{Qt\+Bool\+Property\+Manager}{p.}{classQtBoolPropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Bool\+Property\+Manager()}{p.}{classQtFlagPropertyManager_adb39f20e04ec53278a2d39f871c411b3} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

In addition, \doxyref{Qt\+Flag\+Property\+Manager}{p.}{classQtFlagPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtFlagPropertyManager_ac503903cd623bd5b2a02883b490b0859} signal which is emitted whenever a property created by this manager changes, and the \doxyref{flag\+Names\+Changed()}{p.}{classQtFlagPropertyManager_a5392ae06be8dee42100e8a159dadd135} signal which is emitted whenever the list of flag names is altered.
