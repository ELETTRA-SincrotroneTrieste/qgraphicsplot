\subsection{Detailed Description}
A font property has nested {\itshape family}, {\itshape point\+Size}, {\itshape bold}, {\itshape italic}, {\itshape underline}, {\itshape strike\+Out} and {\itshape kerning} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtFontPropertyManager_add1bfd953d670b0c3e78ec5ae4fc1128} function, and set using the \doxyref{set\+Value()}{p.}{classQtFontPropertyManager_a3c6874b1134907b493b26fe8d9b42a64} slot.

The subproperties are created by \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager}, \doxyref{Qt\+Enum\+Property\+Manager}{p.}{classQtEnumPropertyManager} and \doxyref{Qt\+Bool\+Property\+Manager}{p.}{classQtBoolPropertyManager} objects. These managers can be retrieved using the corresponding \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtFontPropertyManager_a21f960d24c80e58c9e7f28e283086441}, \doxyref{sub\+Enum\+Property\+Manager()}{p.}{classQtFontPropertyManager_adcd8bb28c1b2bf8255d87e65800e61c8} and \doxyref{sub\+Bool\+Property\+Manager()}{p.}{classQtFontPropertyManager_a2261455e4f3490c1b153c16331044213} functions. In order to provide editing widgets for the subproperties in a property browser widget, these managers must be associated with editor factories.

In addition, \doxyref{Qt\+Font\+Property\+Manager}{p.}{classQtFontPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtFontPropertyManager_a6e316ac371b5b03f203b3a0581be3c3a} signal which is emitted whenever a property created by this manager changes.
