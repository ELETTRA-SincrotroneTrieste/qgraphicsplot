\subsection{Detailed Description}
An int property has a current value, and a range specifying the valid values. The range is defined by a minimum and a maximum value.

The property\textquotesingle{}s value and range can be retrieved using the \doxyref{value()}{p.}{classQtIntPropertyManager_abc2262d297496aff50d8cafa3e90b0ff}, \doxyref{minimum()}{p.}{classQtIntPropertyManager_acf24786c2e0990165cc69349d6acc580} and \doxyref{maximum()}{p.}{classQtIntPropertyManager_a3277dc219e7689a0656420f472835ff1} functions, and can be set using the \doxyref{set\+Value()}{p.}{classQtIntPropertyManager_af918e1e91c69de900a3ce3fe8fb4291f}, \doxyref{set\+Minimum()}{p.}{classQtIntPropertyManager_abb419f45a387832a3b74f8c9180dd176} and \doxyref{set\+Maximum()}{p.}{classQtIntPropertyManager_aa09805459434ad6e58274c8213293f89} slots. Alternatively, the range can be defined in one go using the \doxyref{set\+Range()}{p.}{classQtIntPropertyManager_a313c2054fa57dda4e39bfaa6ee64cc38} slot.

In addition, \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtIntPropertyManager_a524990cc950be2174854c0150f906776} signal which is emitted whenever a property created by this manager changes, and the \doxyref{range\+Changed()}{p.}{classQtIntPropertyManager_a2e89b90fb975144bad068d9eee7c2095} signal which is emitted whenever such a property changes its range of valid values.
