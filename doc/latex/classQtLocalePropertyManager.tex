\subsection{Detailed Description}
A locale property has nested {\itshape language} and {\itshape country} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtLocalePropertyManager_abded91083a69cd875f9aa585b0c808cd} function, and set using the \doxyref{set\+Value()}{p.}{classQtLocalePropertyManager_acc0660fa1d6e9bb101b8617ad586f519} slot.

The subproperties are created by \doxyref{Qt\+Enum\+Property\+Manager}{p.}{classQtEnumPropertyManager} object. These submanager can be retrieved using the \doxyref{sub\+Enum\+Property\+Manager()}{p.}{classQtLocalePropertyManager_a340e2726c6b1d9e46d7e32bc594bfcb2} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with editor factory.

In addition, \doxyref{Qt\+Locale\+Property\+Manager}{p.}{classQtLocalePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtLocalePropertyManager_a19237ddbcec233ecb7d68549398650bf} signal which is emitted whenever a property created by this manager changes.
