\subsection{Detailed Description}
A point property has nested {\itshape x} and {\itshape y} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtPointFPropertyManager_ae875d0712ffd73978cd8220e3eef1960} function, and set using the \doxyref{set\+Value()}{p.}{classQtPointFPropertyManager_a6a6c35723f2eb0598052a29078f78a18} slot.

The subproperties are created by a \doxyref{Qt\+Double\+Property\+Manager}{p.}{classQtDoublePropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Double\+Property\+Manager()}{p.}{classQtPointFPropertyManager_a682126563042030b9ba4b370d8f19b05} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

In addition, \doxyref{Qt\+Point\+F\+Property\+Manager}{p.}{classQtPointFPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtPointFPropertyManager_a30f3c6bcce08c1d011bfa3fafe3bff10} signal which is emitted whenever a property created by this manager changes.
