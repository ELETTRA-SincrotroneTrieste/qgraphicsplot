\subsection{Detailed Description}
A point property has nested {\itshape x} and {\itshape y} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtPointPropertyManager_a6f617d39fc178ee6a5e4d12a9679ed4a} function, and set using the \doxyref{set\+Value()}{p.}{classQtPointPropertyManager_afcb764ec969c22f0131ac1151d669896} slot.

The subproperties are created by a \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtPointPropertyManager_a4d0cb1a2119f0980e502976162533b15} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

In addition, \doxyref{Qt\+Point\+Property\+Manager}{p.}{classQtPointPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtPointPropertyManager_aa3a62c11c816a93684449ae1128ff933} signal which is emitted whenever a property created by this manager changes.
