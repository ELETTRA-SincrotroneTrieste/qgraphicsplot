\subsection{Detailed Description}
Properties are created by objects of \doxyref{Qt\+Abstract\+Property\+Manager}{p.}{classQtAbstractPropertyManager} subclasses; a manager can create properties of a given type, and is used in conjunction with the \doxyref{Qt\+Abstract\+Property\+Browser}{p.}{classQtAbstractPropertyBrowser} class. A property is always owned by the manager that created it, which can be retrieved using the \doxyref{property\+Manager()}{p.}{classQtProperty_a7ad6143ada1595b7b304c3324568f31e} function.

\doxyref{Qt\+Property}{p.}{classQtProperty} contains the most common property attributes, and provides functions for retrieving as well as setting their values\+:

Getter  Setter   \doxyref{property\+Name()}{p.}{classQtProperty_ac15755ee28f7718b44121a1365b39a8a}  \doxyref{set\+Property\+Name()}{p.}{classQtProperty_ade8c6cab7fca8e29f3d60f0a41aaed99}   \doxyref{status\+Tip()}{p.}{classQtProperty_a91197c2ee6dfe4446f33594840725bba}  \doxyref{set\+Status\+Tip()}{p.}{classQtProperty_aec97d22e2483272a7678d1be7f5d43e4}   \doxyref{tool\+Tip()}{p.}{classQtProperty_ade424dd4dd39a76941ee5ea8147296f2}  \doxyref{set\+Tool\+Tip()}{p.}{classQtProperty_a32d231f240e359f38e443f2823d28418}   \doxyref{whats\+This()}{p.}{classQtProperty_a1501ee79830b5aa9fd376adf1e67ed05}  \doxyref{set\+Whats\+This()}{p.}{classQtProperty_af0754f48172eabda6c7fc88c841b9dda}   \doxyref{is\+Enabled()}{p.}{classQtProperty_afe9f875bfa727f3270dc1d1a6d675872}  \doxyref{set\+Enabled()}{p.}{classQtProperty_a7532b98f02bb643211b946a13bca4153}   \doxyref{is\+Modified()}{p.}{classQtProperty_aecd8c5c810a69bf3d7659476f0088358}  \doxyref{set\+Modified()}{p.}{classQtProperty_a04e7e0f85291138532afd402267a0cb1}   \doxyref{value\+Text()}{p.}{classQtProperty_a22b03edfd4b9cb8cc39c9c7af3390ebc}  Nop   \doxyref{value\+Icon()}{p.}{classQtProperty_ae1be1b52a375a5b781bc938f866da163}  Nop 

It is also possible to nest properties\+: \doxyref{Qt\+Property}{p.}{classQtProperty} provides the \doxyref{add\+Sub\+Property()}{p.}{classQtProperty_a48af72160de86b566253b19098143c53}, \doxyref{insert\+Sub\+Property()}{p.}{classQtProperty_a8e47b08ff4e980d46fb5a8ee71f347bd} and \doxyref{remove\+Sub\+Property()}{p.}{classQtProperty_aaed9046cdd44964eb859d7c41794a4b4} functions to manipulate the set of subproperties. Use the \doxyref{sub\+Properties()}{p.}{classQtProperty_aaf95c11630e0aae1f026a28cac7d5578} function to retrieve a property\textquotesingle{}s current set of subproperties. Note that nested properties are not owned by the parent property, i.\+e. each subproperty is owned by the manager that created it.
