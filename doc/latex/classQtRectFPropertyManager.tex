\subsection{Detailed Description}
A rectangle property has nested {\itshape x}, {\itshape y}, {\itshape width} and {\itshape height} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtRectFPropertyManager_ab3931343e4b8d3c784f99f5607b40d95} function, and set using the \doxyref{set\+Value()}{p.}{classQtRectFPropertyManager_afa348e09778f9490586e13135b5843f1} slot.

The subproperties are created by a \doxyref{Qt\+Double\+Property\+Manager}{p.}{classQtDoublePropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Double\+Property\+Manager()}{p.}{classQtRectFPropertyManager_acb841ec2f20a57a8ed7ec7b329bf9252} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

A rectangle property also has a constraint rectangle which can be retrieved using the \doxyref{constraint()}{p.}{classQtRectFPropertyManager_a24f321a444aa0214719ba8622ad1d2e5} function, and set using the \doxyref{set\+Constraint()}{p.}{classQtRectFPropertyManager_ad287f58f48260a00de5d29e7f6252c53} slot.

In addition, \doxyref{Qt\+Rect\+F\+Property\+Manager}{p.}{classQtRectFPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtRectFPropertyManager_a865fc49f5579fd737103dc3c7fd4ea48} signal which is emitted whenever a property created by this manager changes, and the \doxyref{constraint\+Changed()}{p.}{classQtRectFPropertyManager_a0761a9cb651208ea3a4c907ade062ff1} signal which is emitted whenever such a property changes its constraint rectangle.
