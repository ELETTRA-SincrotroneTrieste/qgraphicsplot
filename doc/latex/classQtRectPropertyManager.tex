\subsection{Detailed Description}
A rectangle property has nested {\itshape x}, {\itshape y}, {\itshape width} and {\itshape height} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtRectPropertyManager_a858f9fc7e0aa01612f8d78258ef6c8a7} function, and set using the \doxyref{set\+Value()}{p.}{classQtRectPropertyManager_a407029ec29077e974452c1a3318cb261} slot.

The subproperties are created by a \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtRectPropertyManager_a58fe4307cd200083b6315d171a39b88c} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

A rectangle property also has a constraint rectangle which can be retrieved using the \doxyref{constraint()}{p.}{classQtRectPropertyManager_aa68218abec0dfc12814813df7a9f12d6} function, and set using the \doxyref{set\+Constraint()}{p.}{classQtRectPropertyManager_a1c3f251de141136dbd5feafd485652b1} slot.

In addition, \doxyref{Qt\+Rect\+Property\+Manager}{p.}{classQtRectPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtRectPropertyManager_af1aff19ae99196778d6b545ba01908f0} signal which is emitted whenever a property created by this manager changes, and the \doxyref{constraint\+Changed()}{p.}{classQtRectPropertyManager_a4de6d6274fb1124a2d8f71e3436645e0} signal which is emitted whenever such a property changes its constraint rectangle.
