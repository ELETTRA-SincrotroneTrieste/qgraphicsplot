\subsection{Detailed Description}
A size policy property has nested {\itshape horizontal\+Policy}, {\itshape vertical\+Policy}, {\itshape horizontal\+Stretch} and {\itshape vertical\+Stretch} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtSizePolicyPropertyManager_ad15373c73eb67f7c7427b71ba86f043b} function, and set using the \doxyref{set\+Value()}{p.}{classQtSizePolicyPropertyManager_a072ee2c346fc0097ec01adb1ac156cbe} slot.

The subproperties are created by \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} and \doxyref{Qt\+Enum\+Property\+Manager}{p.}{classQtEnumPropertyManager} objects. These managers can be retrieved using the \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtSizePolicyPropertyManager_a9472041448973fc846dc137bb8183de6} and \doxyref{sub\+Enum\+Property\+Manager()}{p.}{classQtSizePolicyPropertyManager_aa31c1c1ec653bf2fab4af34ddfc0f0f0} functions respectively. In order to provide editing widgets for the subproperties in a property browser widget, these managers must be associated with editor factories.

In addition, \doxyref{Qt\+Size\+Policy\+Property\+Manager}{p.}{classQtSizePolicyPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtSizePolicyPropertyManager_a1959382958a9b966a590607804d81644} signal which is emitted whenever a property created by this manager changes.
