\subsection{Detailed Description}
A size property has nested {\itshape width} and {\itshape height} subproperties. The top-\/level property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtSizePropertyManager_a6cac477aebb79887409152e9a6005bf2} function, and set using the \doxyref{set\+Value()}{p.}{classQtSizePropertyManager_a61be90b7c37d4c5194cfc469f43b20a7} slot.

The subproperties are created by a \doxyref{Qt\+Int\+Property\+Manager}{p.}{classQtIntPropertyManager} object. This manager can be retrieved using the \doxyref{sub\+Int\+Property\+Manager()}{p.}{classQtSizePropertyManager_a2b6bf7c0605312bd501cf2553f22adf6} function. In order to provide editing widgets for the subproperties in a property browser widget, this manager must be associated with an editor factory.

A size property also has a range of valid values defined by a minimum size and a maximum size. These sizes can be retrieved using the \doxyref{minimum()}{p.}{classQtSizePropertyManager_a603e4d3d162ddb23e91374191b84f68d} and the \doxyref{maximum()}{p.}{classQtSizePropertyManager_aeea0240d2b2f9aa5c5d2774ac10f9779} functions, and set using the \doxyref{set\+Minimum()}{p.}{classQtSizePropertyManager_aa476ac5c003190f9d9c5fc2c6ad8ecde} and \doxyref{set\+Maximum()}{p.}{classQtSizePropertyManager_ae61e1d1ace1aa09791d1eeed92489a80} slots. Alternatively, the range can be defined in one go using the \doxyref{set\+Range()}{p.}{classQtSizePropertyManager_ae33ffb20e71a6c0d543650ce60ec529c} slot.

In addition, \doxyref{Qt\+Size\+Property\+Manager}{p.}{classQtSizePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtSizePropertyManager_a747c0437bcba80a49393f58489f9a9bc} signal which is emitted whenever a property created by this manager changes, and the \doxyref{range\+Changed()}{p.}{classQtSizePropertyManager_ade832b6b1b40c086b73771f8f8a3a5c4} signal which is emitted whenever such a property changes its range of valid sizes.
