\subsection{Detailed Description}
A string property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtStringPropertyManager_a1f16fe96955a990e72cb3124e3e034be} function, and set using the \doxyref{set\+Value()}{p.}{classQtStringPropertyManager_afa1d5070c04ece70e5908ff4234484a4} slot.

The current value can be checked against a regular expression. To set the regular expression use the \doxyref{set\+Reg\+Exp()}{p.}{classQtStringPropertyManager_a19cdcabc7a14202613a11f3a8bd79805} slot, use the \doxyref{reg\+Exp()}{p.}{classQtStringPropertyManager_ad29ad977bc2832a98f0ab76535397c78} function to retrieve the currently set expression.

In addition, \doxyref{Qt\+String\+Property\+Manager}{p.}{classQtStringPropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtStringPropertyManager_a39f5e49a80f66a410eeb20c762436a82} signal which is emitted whenever a property created by this manager changes, and the \doxyref{reg\+Exp\+Changed()}{p.}{classQtStringPropertyManager_afe2957a9746d51feb7136692966b258c} signal which is emitted whenever such a property changes its currently set regular expression.
