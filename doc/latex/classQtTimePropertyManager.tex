\subsection{Detailed Description}
A time property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtTimePropertyManager_a264573824c0fb805e924097bd1b3ea16} function, and set using the \doxyref{set\+Value()}{p.}{classQtTimePropertyManager_a7e3b29bff17c5210c7cb7cebecfa1ec2} slot.

In addition, \doxyref{Qt\+Time\+Property\+Manager}{p.}{classQtTimePropertyManager} provides the \doxyref{value\+Changed()}{p.}{classQtTimePropertyManager_a124666acfe564a8ef3ce333b11108d8f} signal which is emitted whenever a property created by this manager changes.
