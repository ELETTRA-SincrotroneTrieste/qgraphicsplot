\subsection{Detailed Description}
\doxyref{Qt\+Variant\+Property}{p.}{classQtVariantProperty} provides additional A\+PI\+: A property\textquotesingle{}s type, value type, attribute values and current value can easily be retrieved using the \doxyref{property\+Type()}{p.}{classQtVariantProperty_a61984558f141a6e63d9f0e4e05ae86f3}, \doxyref{value\+Type()}{p.}{classQtVariantProperty_a2350051e47036111c188f914fffc85ae}, \doxyref{attribute\+Value()}{p.}{classQtVariantProperty_a293f3e6b57ad40647b7cb41441d99357} and \doxyref{value()}{p.}{classQtVariantProperty_ac75f95a1a93d510e5920fa7733647320} functions respectively. In addition, the attribute values and the current value can be set using the corresponding \doxyref{set\+Value()}{p.}{classQtVariantProperty_aec9dbb18fc48903f6c217a8c3b9a3e69} and \doxyref{set\+Attribute()}{p.}{classQtVariantProperty_a906f63c5230a99ec0ccefc872ee83de8} functions.

For example, instead of writing\+:


\begin{DoxyCode}
QtVariantPropertyManager *variantPropertyManager;
QtProperty *property;

variantPropertyManager->setValue(property, 10);
\end{DoxyCode}


you can write\+:


\begin{DoxyCode}
QtVariantPropertyManager *variantPropertyManager;
QtVariantProperty *property;

\textcolor{keyword}{property}->setValue(10);
\end{DoxyCode}


\doxyref{Qt\+Variant\+Property}{p.}{classQtVariantProperty} instances can only be created by the \doxyref{Qt\+Variant\+Property\+Manager}{p.}{classQtVariantPropertyManager} class.
