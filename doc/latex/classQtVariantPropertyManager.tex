\subsection{Detailed Description}
\doxyref{Qt\+Variant\+Property\+Manager}{p.}{classQtVariantPropertyManager} provides the \doxyref{add\+Property()}{p.}{classQtVariantPropertyManager_a82c785789a85a4e037ab34f19f786486} function which creates \doxyref{Qt\+Variant\+Property}{p.}{classQtVariantProperty} objects. The \doxyref{Qt\+Variant\+Property}{p.}{classQtVariantProperty} class is a convenience class handling Q\+Variant based properties inheriting \doxyref{Qt\+Property}{p.}{classQtProperty}. A \doxyref{Qt\+Property}{p.}{classQtProperty} object created by a \doxyref{Qt\+Variant\+Property\+Manager}{p.}{classQtVariantPropertyManager} instance can be converted into a \doxyref{Qt\+Variant\+Property}{p.}{classQtVariantProperty} object using the \doxyref{variant\+Property()}{p.}{classQtVariantPropertyManager_ab1037399e3f3b651f3bd6375cdd8bec1} function.

The property\textquotesingle{}s value can be retrieved using the \doxyref{value()}{p.}{classQtVariantPropertyManager_ad0852bb5f326ee74da5396f025975c5b}, and set using the \doxyref{set\+Value()}{p.}{classQtVariantPropertyManager_ad3cc8978018b8f6932b39f1bb8db9371} slot. In addition the property\textquotesingle{}s type, and the type of its value, can be retrieved using the \doxyref{property\+Type()}{p.}{classQtVariantPropertyManager_afb992fbd8634064b153452ebe5a6bbd6} and \doxyref{value\+Type()}{p.}{classQtVariantPropertyManager_a9ae282dab0a10e13ddbe8f15e1ac6cea} functions respectively.

A property\textquotesingle{}s type is a Q\+Variant\+::\+Type enumerator value, and usually a property\textquotesingle{}s type is the same as its value type. But for some properties the types differ, for example for enums, flags and group types in which case \doxyref{Qt\+Variant\+Property\+Manager}{p.}{classQtVariantPropertyManager} provides the \doxyref{enum\+Type\+Id()}{p.}{classQtVariantPropertyManager_a98a1551f0e64e3cd1046c752b95e1e24}, \doxyref{flag\+Type\+Id()}{p.}{classQtVariantPropertyManager_afcc62378daeb0426889a5b91d63dd524} and \doxyref{group\+Type\+Id()}{p.}{classQtVariantPropertyManager_a5192bc3d88c56894ae14442e29c949f9} functions, respectively, to identify their property type (the value types are Q\+Variant\+::\+Int for the enum and flag types, and Q\+Variant\+::\+Invalid for the group type).

Use the \doxyref{is\+Property\+Type\+Supported()}{p.}{classQtVariantPropertyManager_a3174afaf91b0048ae7e9baf0d52c6c82} function to check if a particular property type is supported. The currently supported property types are\+:

The attributes for a given property type can be retrieved using the \doxyref{attributes()}{p.}{classQtVariantPropertyManager_a2a0d1f4d5f3955128acd50a58668b34a} function. Each attribute has a value type which can be retrieved using the \doxyref{attribute\+Type()}{p.}{classQtVariantPropertyManager_a8bf265fc0e22bc20f813570ab618da36} function, and a value accessible through the \doxyref{attribute\+Value()}{p.}{classQtVariantPropertyManager_a3903f68e029def8d555c6e27b58a84e1} function. In addition, the value can be set using the \doxyref{set\+Attribute()}{p.}{classQtVariantPropertyManager_a55adf195cb65c4bab2aee1ee8d973582} slot.

Qt\+Variant\+Manager also provides the \doxyref{value\+Changed()}{p.}{classQtVariantPropertyManager_a255e8def63d723ae2573041e3090d00e} signal which is emitted whenever a property created by this manager change, and the \doxyref{attribute\+Changed()}{p.}{classQtVariantPropertyManager_a8440543d9eeda9d21c587c92399dc2c9} signal which is emitted whenever an attribute of such a property changes.
