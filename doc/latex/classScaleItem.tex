\doxysubsection{Detailed Description}
\begin{DoxyParagraph}{Description}
The \doxyref{Scale\+Item}{p.}{classScaleItem} is a Q\+Graphics\+Object that draws and axis scale on a \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}. What a \doxyref{Scale\+Item}{p.}{classScaleItem} draws is\+: \begin{DoxyItemize}
\item a line for the backbone; \item ticks on the backbone \item labels on each tick, when there is enough space to draw the text.\end{DoxyItemize}

\end{DoxyParagraph}
\begin{DoxyParagraph}{Position of the axes}
The axes are placed by \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} through the classes \doxyref{Axes\+Manager}{p.}{classAxesManager} and \doxyref{Axis\+Couple}{p.}{classAxisCouple}. Each \doxyref{Axis\+Couple}{p.}{classAxisCouple} in the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} represents a couple of axis, one with Horizontal orientation and the other with Vertical orientation. The default positioning of the axes is done by default such as each the axes cross each other in the middle. If you want to change the reciprocal axes position, see \doxyref{Plot\+Scene\+Widget\+::set\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_aea844334c4b2e71ff6eba2f57cd562f9} \doxyref{Plot\+Scene\+Widget\+::set\+Default\+X\+Axis\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_a0c9b92b6a5aca6da0fa85b9f9b7aca32} and \doxyref{Plot\+Scene\+Widget\+::set\+Default\+Y\+Axis\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_a759d14c7e662ce3afabebd7dcf167d9c}.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Relevant axis properties}
\begin{DoxyItemize}
\item tick\+Step\+Len the distance between two subsequent ticks, in scale coordinates. A tick step length of one on the x axis means one tick every increment of one on the x value If tick\+Step\+Len is set to -\/1, then the ticks distance is automatically chosen. \item axis\+Labels\+Enabled enables or disables the text labels on the axis. \item grid\+Enabled shows or hides the grid on the plot \item axis\+Labels\+Format the format of the number to display on the axis (it\textquotesingle{}s printf() format) \item upper\+Bound sets the upper bound on the scale item. axis\+Autoscale\+Enabled must be set to false \item lower\+Bound sets the lower bound on the scale item. axis\+Autoscale\+Enabled must be set to false \item axis\+Autoscale\+Enabled enables or disables auto scaling. \end{DoxyItemize}

\end{DoxyParagraph}

The \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} creates two \doxyref{Scale\+Item}{p.}{classScaleItem} objects for you, an x item and an y item. They are laid out so that they cross each other in the middle (see \doxyref{Plot\+Scene\+Widget\+::set\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_aea844334c4b2e71ff6eba2f57cd562f9}, \doxyref{Plot\+Scene\+Widget\+::set\+Default\+X\+Axis\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_a0c9b92b6a5aca6da0fa85b9f9b7aca32} and \doxyref{Plot\+Scene\+Widget\+::set\+Default\+Y\+Axis\+Origin\+Pos\+Percentage}{p.}{classPlotSceneWidget_a759d14c7e662ce3afabebd7dcf167d9c})

If you want to customize the scale labels, then subclass \doxyref{Scale\+Label\+Interface}{p.}{classScaleLabelInterface} and provide a method named label in order to tell the \doxyref{Scale\+Item}{p.}{classScaleItem} what text to draw in correspondance to a specific value on the axis. \begin{DoxySeeAlso}{See also}
\doxyref{Scale\+Label\+Interface}{p.}{classScaleLabelInterface}
\end{DoxySeeAlso}
The \doxyref{Scale\+Item}{p.}{classScaleItem} is a Q\+Graphics\+Object which is added to the scene of the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}.

The \doxyref{Scale\+Item\+::\+Id}{p.}{classScaleItem_a2014770ca14c8e22d594ba7b3f3b379b} is an enum with a bunch of default items. \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} manages the two default axes for you. If you want to add more axes, please have a look to the \doxyref{Scale\+Item\+::\+Id}{p.}{classScaleItem_a2014770ca14c8e22d594ba7b3f3b379b} enum and use one of the predefined values or add others by yourself.
