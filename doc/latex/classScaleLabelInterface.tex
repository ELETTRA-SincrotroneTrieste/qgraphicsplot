\doxysubsection{Detailed Description}
To customize axis labels, just provide an implementation of this \doxyref{Scale\+Label\+Interface}{p.}{classScaleLabelInterface} and install it on a \doxyref{Scale\+Item}{p.}{classScaleItem}. You must implement the label method, that returns a Q\+String depending on the value of the point in the axis.

\begin{DoxyParagraph}{Example}
A custom implementation may return a string displaying a date/time value represented by a timestamp on the x axis. 
\end{DoxyParagraph}
